# FocusForge

A simple study planner built with JavaScript, Express.js, and Tailwind CSS.

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Dark mode
- Add due date for tasks
- Improved UI design
- Unique task IDs

## Tech Stack

- HTML
- Tailwind CSS
- JavaScript
- Node.js
- Express.js


## Project Status
 Kanban board feature is currently in development.
