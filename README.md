# FocusForge

FocusForge is a productivity-focused study planner built with **JavaScript, Express.js, MongoDB, and Tailwind CSS**. It helps students organize tasks, stay focused, and build consistent study habits through task management, focus sessions, progress tracking, and productivity analytics.

## Features

### User Authentication

* User registration and login
* Session-based authentication using Passport.js
* Protected dashboard for authenticated users

### Task Management

* Create, edit, and delete tasks
* Assign due dates
* Mark tasks as completed
* Store tasks in MongoDB

### Dashboard

* Today's Tasks
* Upcoming Tasks
* Weekly progress chart
* Recently Completed Tasks
* Real-time statistics:

  * Total Tasks
  * Completed Tasks
  * Pending Tasks
  * Current Study Streak
  * Total Focus Sessions

### Focus Timer

* Pomodoro-style focus timer
* Track completed focus sessions
* Automatically store sessions in MongoDB
* Real-time session updates

### Study Streak

* Tracks consecutive study days
* Automatically resets after missed days
* Displays current streak on the dashboard

### Weekly Progress

* Interactive productivity chart using Chart.js
* Visualizes completed tasks throughout the week

### Kanban Board

* To Do
* In Progress
* Done

## Tech Stack

* HTML
* Tailwind CSS
* JavaScript
* Node.js
* Express.js
* MongoDB
* Mongoose
* Passport.js
* Chart.js

FocusForge is actively under development. Future improvements include notifications, enhanced productivity analytics, and additional study-focused features.
